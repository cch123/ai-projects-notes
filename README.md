## 组织方式

每个项目一个顶层目录，内部一般包含：

- `README.md`——项目总览、核心概念、阅读路径
- `architecture.md`——分层、依赖、关键流程
- `notes/`——逐模块、逐文件的深入笔记（按需添加）
- `snippets/`——源码摘录与注释（按需添加）
